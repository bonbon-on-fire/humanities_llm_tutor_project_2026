# Batch Type 03: Different Persona + Same Version + Same Exercise

This batch type groups transcripts that share exercise and version but differ in student persona:
- **Different student personas** (e.g., `chaotic_01`, `cooperative_01`, `clueless_01`)
- **Same persona version** (e.g., all version `01`)
- **Same exercise** (e.g., all `philosophy/exercise_01`)

Tests persona differentiation and judge ability to distinguish between different student types while applying appropriate pedagogical standards. Reveals judge sophistication in understanding student archetypes, bias detection across persona types, and whether scoring appropriately accounts for individual learning challenges and strengths.
